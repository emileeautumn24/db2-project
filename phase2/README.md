## Installation & Setup

To run this project locally, we'll need XAMPP.

Move Project Files: Place the db2-project folder into your XAMPP directory:

C:\xampp\htdocs\db2-project\

Start Services: Open the XAMPP Control Panel and start Apache and MySQL.

Database Setup:

Open phpMyAdmin.

Create a new database named db2.

Import/Copy the SQL files located in phase2/sql/ in this order:

DB2-tables.sql (Schema)

DB2-data.sql (Our Data)

Access the App: Navigate to http://localhost/db2-project/phase2/login.html in our browser.

## Directory Structure

